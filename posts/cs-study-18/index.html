[CS 스터디] 복잡도

2026-06-22

1. 복잡도와 빅오 표기법

  • 알고리즘을 수행할 때 자원을 낭비하지 않으려면 효율적으로 짜야 함
  • 이때 얼마나 효율적인지 정량화하는 데 시간 복잡도와 공간 복잡도를 사용함
    • 시간 복잡도: 알고리즘의 실행 시간을 정량화
    • 공간 복잡도: 실행하는 데 필요한 메모리 사용량을 정량화

빅오(Big-O) 표기법

  • 입력 값(n)에 대한 수식에서 최고차항을 기준으로 표현한 최악의 시간 복잡도
  • 계수와 상수항은 무시하고 최고차항만으로 표현
    • 예) 2n² + 3n + 1 → O(n²)

점근적 표기법

  • 입력 크기가 충분히 클 때의 증가율을 기준으로 복잡도를 표현하는 방법
표기법 의미
빅오(O) 최악의 경우
빅세타(Θ) 평균의 경우
빅오메가(Ω) 최선의 경우

대표적인 시간 복잡도 (빠른 순)

복잡도 명칭 예시
O(1) 상수 배열 인덱스 접근
O(log n) 로그 이진 탐색
O(n) 선형 단순 반복문 순회
O(n log n) 선형 로그 병합 정렬, 퀵 정렬(평균)
O(n²) 이차 이중 반복문
O(2ⁿ) 지수 부분집합 완전탐색
O(n!) 팩토리얼 순열 완전탐색
